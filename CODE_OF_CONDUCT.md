# Code of Conduct

PondiFarmApp is a project of Euro Union Consult. We expect everyone working in this repository — employees, contractors, partners, students, and external collaborators — to interact in a professional, respectful, and constructive way.

## Our standards

We ask all participants to:

- Be respectful in code review, issues, discussions, and any other channel related to this project.
- Assume good intent and ask questions before assuming a mistake is deliberate.
- Provide feedback that is concrete, actionable, and focused on the work, not the person.
- Credit the work of others and acknowledge prior contributions.
- Keep discussions focused on the project; private disagreements should be resolved off-channel.
- Treat all collaborators equally, regardless of role, seniority, background, or institution.

## What is not acceptable

- Personal attacks, insults, or sustained disruption of discussions.
- Sharing private information of other participants without consent.
- Any form of conduct that a reasonable professional would consider inappropriate in a workplace setting.

## Scope

This Code of Conduct applies to all project spaces: the GitHub repository (issues, pull requests, discussions, code review), project email threads, video calls, in-person meetings related to the project, and any other forum where someone is representing the project.

## Reporting

If you observe or experience a breach of this Code of Conduct, please contact the project coordinator at Euro Union Consult: `projects@eurounionconsult.com`.

Reports are handled in confidence. The reporter is not required to take part in any decision about the response.

## Enforcement

The project coordinator at Euro Union Consult is responsible for clarifying this Code of Conduct and deciding on the response to any reported incident. Possible responses include a private conversation, a written warning, temporary or permanent removal of access to the repository, or termination of a contributor agreement.

## Acknowledgement

This Code of Conduct is adapted from common open-source community guidelines and tailored to the working context of Euro Union Consult.
